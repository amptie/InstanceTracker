InstanceTracker by amptie:<br>
<br>
Shows you 5-man dungeon IDs to track your Lockouts on Vanilla 1.12<br>
<br>
Commands:<br>
/it show/hide - Shows/Hides the button for the tracking <br>
/it clear - Resets all active IDs <br>
/it list - Shows active IDs in the chat <br>
/it resetbtn - Resets the button <br>
<br>
<br>
Mouseover the button and you will see the active IDs with the timers written in MM:SS<br>
Language is German only at the moment<br>
<br>
Dungeons IDs will be shown account-wide<br>
