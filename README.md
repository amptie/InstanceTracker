InstanceTracker by amptie:

Shows you 5-man dungeon IDs to track your Lockouts on Vanilla 1.12

Commands:

/it show/hide - Shows/Hides the button for the tracking
/it clear - Resets all active IDs
/it list - Shows active IDs in the chat
/it resetbtn - Resets the button

Mouseover the button and you will see the active IDs with the timers written in MM:SS
Language is German only at the moment

Dungeons IDs will be shown account-wide
