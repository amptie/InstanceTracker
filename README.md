InstanceTracker by amptie:<br>
<br>
Shows you 5-man dungeon IDs to track your Lockouts on Vanilla 1.12<br>
<br>
Commands:<br>
/it show/hide - Shows/Hides the button for the tracking <br>
/it clear - Resets all active IDs <br>
/it list - Shows active IDs in the chat <br>
/it resetbtn - Resets the button <br>
<br>
<br>
Mouseover the button and you will see the active IDs with the timers written in MM:SS<br>
Language is German only at the moment<br>
<br>
Dungeons IDs will be shown account-wide<br>

How does it work?<br>
Since there is no 5-man Dungeon ID in Vanilla 1.12 this Addon is triggered by Dungeon Zones and a Loading Screen (when you enter the dungeon).<br>
Because some dungeons have areas in front of them, which are outside but have the same zone name, the Addon will ask you if it should start a new ID, when you enter a dungeon.<br>
This is to prevent starting IDs, when reloading the UI, while in such a pre-dungeon area.<br>
With "Ja" you confirm that you want to start a new ID timer, with "Nein" you deny the start of a new timer<br>
